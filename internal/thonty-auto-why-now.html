Implementation
Build This
1
Load lead list into Airtable with domains + GBP links
Create a base with lead name, domain, GBP URL, and status columns. This is the source of truth for which leads get scanned daily.
2
Build n8n scheduled workflow (daily scan)
Set up a cron-triggered n8n workflow that pulls the active lead list from Airtable each morning and fans out to the data source APIs.
3
Connect DataForSEO for website change detection
For each lead domain, check for significant website changes — redesigns, new pages, broken pages, traffic shifts — that signal they're investing or struggling.
4
Connect GBP API for review monitoring
Pull latest reviews and rating changes for each lead's GBP listing. Flag any rating drop or cluster of negative reviews in the past 7 days.
5
Build Claude prompt for trigger analysis + angle writing
Claude receives the raw signal data, classifies the trigger type, scores urgency, and writes a personalised opening message tied to the specific trigger event.
6
Connect Slack for alert delivery
Format the trigger alert with lead details, trigger type, suggested angle, and action buttons. Post to a dedicated #triggers channel each morning.

Implementation
Build This
1
Connect Fathom webhook to n8n
Set up a webhook trigger in n8n. Point Fathom's post-call webhook at that URL. It fires automatically when a recording is ready.
2
Configure Claude analysis prompt in n8n
The n8n workflow passes the Fathom transcript to Claude via API. The scoring prompt covers all 8 dimensions and instructs Claude to return structured JSON for the Slack formatter.
3
Slack message formatted and delivered
n8n takes Claude's output and posts the formatted coaching brief to your Slack channel. Total time from call end to brief in Slack: under 5 minutes.

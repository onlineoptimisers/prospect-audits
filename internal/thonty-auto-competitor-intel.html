Implementation
Build This
1
Build Slack /competitor command trigger in n8n
Create a Slack slash command that accepts a domain. n8n receives the command via webhook and kicks off the competitive analysis workflow.
2
Connect DataForSEO for competitor domain discovery
Using the lead's domain, pull the top 3 organic competitors in their niche + location. Get domain rating, monthly traffic estimates, and backlink counts for each.
3
Connect SE Ranking for keyword gap analysis
Compare the lead's keyword profile against their top competitors. Identify high-value keywords the competitors rank for that the lead is missing entirely.
4
Build Claude prompt for case study matching + deck narrative
Claude receives the competitive data, matches it against your case study library (niche, starting metrics, city size), and writes the deck narrative connecting the data to your results.
5
Connect Google Drive API for deck creation
Use a branded Google Slides template. Claude's output populates the slides — competitor comparison table, keyword gaps, case study match, and recommended strategy.
6
Log in Airtable with lead name + deck link
Every deck generated gets logged in Airtable with the lead name, domain, generation date, and Drive link. Full audit trail of every call you prepared for.

The best closers don't wing discovery calls. They walk in with more data about the lead's competitors than the lead has themselves. This does that in 5 minutes.
